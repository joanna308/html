Block elements

Block elements start on a new line and are always add some space before and after the element.

Block elements are a little different from inline elements.

Block elements take up all the available width before moving to the next line. It adjusts to the width of the page.

Inline elements

Inline elements do not start on a new line.

They continue on the same line that they are used on.

Inline elements are almost the same of block elements except for the fact that they do not start on a new line

Inline elements take up as much width as necessary unlike block elements that take up the entire width available.

Some examples of inline elements are:

  • span
  • a
  • time
  • input
  • map
  • abbr

For more examples of inline elements look at: inline elements
